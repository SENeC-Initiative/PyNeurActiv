Models module¶

Module containing theoretical or computational models to study or predict the behavior of neuronal populations.

class pyneuractiv.models.Fardet2017_SynchroBurst(parameters, ignore_errors=False, verbose=False)[source]¶

Class computing the theoretical properties of the bursting dynamics for coupled aEIF neurons. Predictions are available for three different synaptic models: “dirac”, “continuous” or “alpha”.

Prepare all the dimensionless values from the dictionnary parameters containing the parameters.

Parameters:
  • parameters (dict) – Dictionary containing the parameters for a model of type aEIF, as defined in the NEST simulator.
  • ignore_errors (bool, optional (default: False)) – Returns NaN without raising an error if the parameters do not lead to synchronous bursting.
  • verbose (bool, optional (default: True)) – Hide warnings if False.

Note

parameters must contain:

  • all neuronal parameters for the NEST aeif model
  • a "model" entry with model name in NEST, e.g. “aeif_psc_alpha”
  • a "weight" entry giving the synaptic strength in pA
  • an "avg_deg" entry giving average number of neighbours
  • a "delay" entry giving the synaptic delay

Compute the following properties of the bursting dynamics:

  • “w_max” (maximum value of w at the end of a burst)
  • “w_min” (minimal value of w juste before a burst starts)

  • “IBI” (inter-burst interval)

time_evolution(model='alpha', num_bursts=3, steps=1000, homogeneous=False, adim=False, show=True)[source]¶

Plot the time evolution of the V and w variables for the theoretical mean-field model.

Parameters:
  • model (str, optional (default: “alpha”)) – Synaptic model, among “dirac”, “continuous” and “alpha”.
  • num_bursts (int, optional (default: 3)) – Number of bursts to model.
  • steps int, optional (default (1000)) – Number of steps taken to model each time interval (initial decrease of V, recovery period and bursting period; default is 1000).
  • homogeneous (bool, optional (default: False)) – Whether the network is considered homogeneous (fixed-in-degree), in which case $w$ is incremented in successive steps of $b$ until $w_{max} > w^*$, or heterogeneous, in which case $w$ increases smoothly up to $w^*$.
  • adim (bool, optional (default: False)) – Whether the variables should be given in dimensionless units.
  • show (bool, optional (default: True)) – Whether the results should be displayed.
Returns:

ts, Vs, ws, stimes (numpy.array objects) – time, voltage, and adaptation variables, as well as the spike times.

class pyneuractiv.models.Simulator_SynchroBurst(num_neurons, parameters, resolution=None, monitor_rate=None, mean_field=True, omp=1, ignore_errors=False, **kwargs)[source]¶

Class computing the theoretical values

Initialize the class:
  • create the network in NEST
  • prepare the dict to contain the results
  • get the state variables that can be recorded
  • create the devices to monitor the spikes and variables
Parameters:
  • num_neurons (tuple) – Number of neurons in the network.
  • parameters (dict) – Parameters of the neurons, same as in Fardet2017_SynchroBurst.
  • resolution (float, optional (default: None)) – Timestep of the simulation in NEST.
  • monitor_rate (float, optional (default: None)) – Sampling frequency for the monitoring device (should be greater or equal to resolution).
  • mean_field (bool, optional (default: True)) – Whether the signals from all neurons should be averaged or kept separated (it is highly recommanded to set it to False to compute phase synchronicity).
  • omp (int, optional (default: 1)) – Number of OpenMP processes to use during the simulation.

See also
